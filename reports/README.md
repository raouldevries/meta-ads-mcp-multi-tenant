# Creative Analysis Reports

This folder contains HTML reports generated by the Creative Analyzer Agent.

## Naming Convention

Reports follow this naming pattern:
```
{type}_analysis_{identifier}_{ad_id}.html
```

Examples:
- `image_analysis_elst_120239386324810661.html`
- `video_analysis_nijmegen_120237318342120381.html`

## Report Types

- **Image Analysis**: Visual composition, color psychology, subject analysis, platform optimization
- **Video Analysis**: Retention curves, subtitle detection, scene analysis, hook effectiveness

## Viewing Reports

Open any HTML file in a browser. Reports are self-contained with embedded CSS and use Font Awesome icons via CDN.
